# Project-YES

***소프트웨어 설계 및 프로젝트*** 과목 프로젝트입니다.  

기존 유튜브에선 실시간 방송들을 분류별로 한 번에 볼 수 없다는 불편함을 해소하고자 해당 아이디어를 선정하게 되었습니다.  

<br/>

 - 현재 인기 있는 생방송들을 한 눈에 볼 수 있습니다.
 - 게임, 뉴스, 먹방 등 분류별로 실시간 방송을 찾을 수 있습니다.
 - 유튜버별 개별 게시판으로 유튜버와 시청자들이 소통 할 수 있습니다.
